# 企业客户邮件模板

> 用途：向企业 HR/培训负责人介绍 TrainHub 平台
> 三个版本：LinkedIn 私信 / 微信消息 / 正式邮件
> 核心卖点：降低采购成本、信息透明可比较、免费咨询无压力
> 注意：企业端不是"卖东西"，而是"帮你省钱"

> 风格：更像朋友推荐，而不是推销

## 发送策略建议

### 目标企业画像

优先触达的企业应满足以下条件：

| 维度 | 标准 | 原因 |
|------|------|------|
| 企业规模 | 100-2000人 | 有培训预算，但没有成熟的内部培训部门 |
| 行业 | 科技、金融、制造、消费品 | 培训需求旺盛，预算充足 |
| 发展阶段 | 快速成长期 | 管理培训需求迫切，但采购经验不足 |
| 决策人 | HR总监、培训经理、学习发展负责人 | 直接的采购决策者或强影响者 |
| 痛点信号 | 最近在招HR/培训岗位、发布过培训相关内容 | 表明正在加大培训投入 |

### 触达渠道优先级

1. **LinkedIn**：专业场景，HR负责人活跃度高
2. **微信**：通过行业社群和活动积累的关系
3. **正式邮件**：适合较大规模企业的正式沟通
4. **HR社群**：在脉脉、HRTech等社群中做价值分享，自然引流

### 触达节奏

| 时间节点 | 动作 |
|---------|------|
| Day 0 | 发送首次消息/邮件 |
| Day 5 | 未回复者发送跟进消息 |
| Day 10 | 分享一篇行业洞察内容（如知乎文章），附带平台链接 |
| Day 20 | 最终跟进，提供具体的培训师推荐（如"您所在行业的XX方向，我们有一位XX背景的培训师"） |
| 之后 | 转为内容触达，不再主动推销 |

### 预期转化率

- 消息打开/阅读率：50%-70%
- 回复率：8%-15%
- 平台访问率：5%-10%
- 咨询转化率：2%-5%

首月目标：触达200家企业HR，获得10-20个咨询。

## 个性化清单

每封邮件/消息必须个性化的信息：

1. **收件人姓名** — 准确
2. **公司名称** — 准确
3. **行业/业务特征** — 表明你了解他们的业务
4. **可能的培训需求** — 根据公司阶段和行业推测
5. **匹配的培训师** — 如果能推荐具体的培训师更好

**核心原则：你不是在推销一个平台，你是在帮他们发现一个更好的采购方式。语气是"分享"和"推荐"，而不是"请求"和"推销"。**
