### 语言风格指南

```
DO（这样写）：
- "一个行业数据让我震惊"
- "你的培训预算有多少真正转化成了交付？"
- "选培训师应该像选餐厅：自己看菜单，自己选"

DON'T（不要这样写）：
- "TrainHub震撼发布！颠覆培训行业！"
- "赶快来体验TrainHub的极致服务！"
- "TrainHub是行业领先的培训师平台"
```

### 核心原则

1. **8:2法则**：80%的内容是行业洞察和实用价值，20%提及平台
2. **教育而非推销**：先帮用户变聪明，再让他们自己做选择
3. **具体而非笼统**：用数据、案例、场景，不用形容词
4. **对话而非宣讲**：像跟一个行业内的朋友聊天，不是发新闻稿
