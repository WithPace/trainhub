# 培训师招募邮件模板

> 用途：邀请自由培训师入驻 TrainHub 平台
> 三个版本：LinkedIn 私信 / 微信消息 / 正式邮件
> 核心卖点：免费入驻、自主定价、直接对接企业、早期入驻特权

---

## 版本一：LinkedIn 私信

> 适用场景：通过 LinkedIn 主动联系培训师
> 字数控制：200字以内，简洁有力
> 发送节奏：第一条不回，3天后发跟进消息

### 首次私信

```
[培训师姓名] 老师您好，

看到您在 [具体领域，如"领导力发展"] 领域的分享，非常认同您关于 [提及对方最近一篇内容/观点] 的观点。

我们在做一件事：TrainHub，一个让培训师直接面对企业客户的平台。
没有中间商加价，培训师自主定价，企业直接看到您的课程和专业背景。

目前平台免费入驻，第一批培训师享受永久免佣金特权。

如果您感兴趣，我可以发一份详细介绍给您。
也可以直接看看：https://withpace.github.io/trainhub/

期待交流。
```

### 跟进消息（3天未回复）

```
[培训师姓名] 老师，冒昧再打扰一下。

补充一个信息：TrainHub 上线后，首批入驻的培训师将获得平台首页推荐位和专属标识。
我们的逻辑很简单——平台起步需要优质培训师，您需要更多企业直接看到您。

如果现在不方便深聊，回复"感兴趣"三个字就好，我把详细资料发您邮箱。
```

## 版本二：微信消息

> 适用场景：通过培训师公开的微信号、或通过共同好友介绍后的首次沟通
> 风格：比 LinkedIn 更亲近，但仍保持专业
> 注意：微信消息不宜太长，分段发送

### 消息一（先建立连接）

```
[培训师姓名] 老师您好！我是 TrainHub 的 [你的名字]。

之前在 [具体场景：知乎/某个论坛/某位朋友推荐] 了解到您在 [领域] 方面的课程，非常认可您的专业功底。

冒昧打扰，想跟您聊一个合作机会，大概2分钟能说清楚，方便吗？
```

### 消息二（等对方回复后，或直接跟发）

```
我们做了一个培训师直连企业的平台——TrainHub。

简单说就是：
- 培训师直接展示课程和背景，企业自己来找来比较
- 不经过机构，没有中间环节加价
- 培训师自己定价，收入100%归您

目前处于早期招募阶段，免费入驻。
第一批入驻的培训师有三个特权：
1. 永久免佣金（后续入驻的培训师会收取少量服务费）
2. 平台首页推荐位
3. "创始培训师"专属标识

您可以先看看平台：https://withpace.github.io/trainhub/

如果觉得方向靠谱，我们可以约个时间详细聊聊入驻流程。很简单，大概15分钟就能搞定。
```

### 消息三（如果培训师提出疑问或顾虑）

```
理解您的顾虑。回答几个常见问题：

Q：入驻后我需要做什么？
A：提供您的专业背景、课程大纲和定价区间就行。我们帮您做好页面展示。后续有企业咨询会直接转给您。

Q：企业客户从哪里来？
A：前期我们通过行业内容（知乎、LinkedIn、公众号）和定向推送获客。第一批客户资源会优先匹配给首批入驻的培训师。

Q：平台怎么盈利？以后会不会收很高的佣金？
A：后续计划收取5%-10%的服务费（对比机构55%-70%的抽成），目前阶段完全免费。我们的承诺是：首批培训师永久免佣金。

Q：会不会跟我现有的机构渠道冲突？
A：不会。TrainHub 是一个增量渠道，帮您接触到那些不通过机构找培训师的企业客户。您现有的合作不受任何影响。
```

## 版本三：正式邮件

> 适用场景：通过公开邮箱联系培训师，或对方要求发正式邮件
> 风格：专业、有温度、不浮夸
> 邮件标题建议：3选1

### 邮件标题（A/B/C 三选一）

- A：`邀请入驻 TrainHub — 让企业直接找到您`
- B：`[培训师姓名] 老师，一个让培训师100%自主定价的平台`
- C：`第一批免佣金名额 — TrainHub 培训师直连平台`

### 邮件正文

```
[培训师姓名] 老师，您好！

我是 TrainHub 平台的 [你的名字]。

之所以联系您，是因为我们在调研 [具体领域] 培训市场时，
注意到您在该领域有 [X] 年的深耕经验和扎实的专业口碑。

先用一句话介绍 TrainHub：
这是一个让培训师直接面对企业客户的平台，没有中间商加价。

—— 为什么做这件事 ——

我们观察到企业培训行业一个长期存在的结构性问题：

企业每年在培训上花费数十万甚至数百万预算，
但其中有50%-70%被中间环节消耗（机构抽成、销售提成、管理费用）。

与此同时，真正上课的培训师只能拿到总费用的30%-50%，
还要花大量精力在获客和维护渠道关系上。

TrainHub 想做的，是重新连接这两端：
让企业直接看到培训师的专业背景、课程内容和真实评价，
让培训师不再依赖中间机构，直接获得企业客户。

—— 入驻 TrainHub 您将获得 ——

1. 专属个人主页
   完整展示您的专业背景、课程体系、定价和客户评价。
   这是您在互联网上的"培训师名片"。

2. 企业客户直连
   有培训需求的企业通过平台发现您、了解您、直接联系您。
   不经过销售中间环节，沟通更高效，需求理解更准确。

3. 100%自主定价
   您的课程，您定价。平台不干预定价策略。

4. 个人品牌积累
   每一次好评、每一个成功案例，都沉淀在您的个人品牌上，
   而不是机构的品牌上。

—— 首批培训师特别权益 ——

作为平台早期阶段，我们正在邀请各领域的优质培训师成为"创始培训师"。
首批入驻的培训师享有以下特权：

  - 永久免佣金（后续入驻的培训师将收取5%-10%服务费）
  - 平台首页推荐位（黄金曝光位置）
  - "创始培训师"专属标识（平台成长后，这个标识会很有辨识度）
  - 优先匹配早期企业客户

—— 入驻流程 ——

极其简单，三步完成：
1. 提供您的专业背景简介（300字左右）
2. 提供1-3门核心课程的大纲和定价区间
3. 我们帮您完成页面搭建，上线前发您确认

整个过程大约需要15-20分钟。
```

## 发送策略建议

### 目标培训师画像

优先邀请的培训师应满足以下条件：

| 维度 | 标准 |
|------|------|
| 从业年限 | 10年以上 |
| 授课经验 | 年均授课100天以上 |
| 专业领域 | 领导力/销售/数字化转型/HR/财务/沟通（六大核心类目） |
| 线上存在感 | 有知乎/LinkedIn/公众号等内容输出 |
| 合作模式 | 自由培训师，不强依赖单一机构 |

### 触达节奏

| 时间节点 | 动作 |
|---------|------|
| Day 0 | 发送首次消息/邮件 |
| Day 3 | 未回复者发送跟进消息 |
| Day 7 | 仍未回复者，通过另一个渠道再次触达（如LinkedIn发过，换微信） |
| Day 14 | 最终跟进，分享平台最新动态（如"又有X位培训师入驻"） |
| Day 14+ | 不再主动打扰，转为内容触达（让对方在知乎/LinkedIn上自然看到平台信息） |

### 预期转化率

基于冷邮件行业基准：
- 打开率预期：40%-60%（精准定向 + 强相关主题行）
- 回复率预期：10%-20%
- 入驻转化率预期：5%-10%

首月目标：触达100位培训师，入驻5-10位。

## 个性化注意事项

每封邮件/消息必须个性化以下信息，绝不群发：

1. **培训师姓名** — 必须准确
2. **具体领域** — 写明对方擅长的方向
3. **内容/观点引用** — 提及对方最近发布的文章、演讲、课程
4. **从业年限** — 体现你做了功课
5. **城市** — 如果知道对方所在城市，可以提及"我们在 [城市] 已有 X 位培训师"

**群发是Permission Marketing的反面。每一封消息都应该让对方感到：你是专门来找我的，不是群发的。**
